🏥 Sistema de web scraping desarrollado en Python para la recopilación y procesamiento de precios de productos farmacéuticos desde múltiples cadenas de farmacias. El proyecto permite automatizar la obtención de datos, normalizar la información recolectada y generar datasets listos para análisis de mercado, monitoreo de competencia y seguimiento de la evolución de precios.

## Tecnologías

- Python
- Requests
- BeautifulSoup
- Pandas
- Selenium
  
## Objetivo

Automatizar la recolección de precios de productos farmacéuticos para facilitar análisis comparativos y monitoreo del mercado.
